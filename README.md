# mini-projeto-python-Vladimir-Gustavo--Franciso-Lemos
Projeto 1: Sistema de Cadastro de Produtos
Um sistema simples de controle de estoque que permite cadastrar, listar, buscar, atualizar e excluir produtos.

Funcionalidades:
- Cadastro com código único, nome, preço, quantidade e categoria.
- Evita códigos duplicados com uso de `set`.
- Categorias definidas em uma `tupla`.
- Armazena produtos em uma `lista` de dicionários.

Projeto 2: Sistema de Controle de Alunos e Notas
Ajuda professores a registrar notas e verificar médias dos alunos.

Funcionalidades:
- Cadastro de alunos com matrícula única.
- Registro de três notas por aluno.
- Cálculo automático da média.
- Relatórios de alunos cadastrados, médias e situação (aprovado/reprovado).

Tecnologias e Conceitos Utilizados
- Estruturas de Repetição (`for`, `while`)
- Estruturas de Dados (`list`, `dict`, `set`, `tuple`)
- Validação de entradas e tratamento de erros (`try/except`)
